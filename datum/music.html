Vars

base_volumeThe base volume to play at, not accounting for fading
does_loopWhether the file loops - If false, the music automatically is removed once it reaches the end
lengthUsed for removing the song if it's non-looping - not necessary to set for loops
priorityHigher priority music plays over lower priority ones. If a looping music is masked by a higher priority music, it will start playing once the higher priority ends if it hasn't been removed.
sound_fileThe file played
